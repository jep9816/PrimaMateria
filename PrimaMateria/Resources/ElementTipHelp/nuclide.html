Nuclides (isotopes)

Index

Forms or species of an element that has the same numbers of protons (hence the same atomic number), but different numbers of neutrons, thus different atomic masses. Technically speaking if an element has only one form that is there is no variability in the number of neutrons that exist in the element, it has no isotopes. There are 21 elements that do not very in the number of neutrons in their nuclei.
