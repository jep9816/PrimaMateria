Tableau Périodique

Un agencement de éléments dans un motif géométrique conçu pour représenter la loi périodique par aligner les éléments sur les périodes et groupes . Éléments avec le même nombre de les orbitales sont dans la même période et éléments avec le même nombre de électrons dans le remplissage orbital et avoir d'autres les propriétés sont dans le même groupe.
