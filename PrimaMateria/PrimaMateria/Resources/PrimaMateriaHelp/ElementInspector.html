Element Inspector Window Help

There are four separate views available to look at element details

  • Atomic Structure

    There are two image views showing element physical properties

    • Crystal Structure

    • Shell Model

  • Element Properties

    There are two views detailing element properties

    • Chemical Properties

    • Physical Properties

  • Nuclides / Isotopes

  • General Info

    By clicking the button, an additional panel, shown below, will be displayed showing additional information about an element from wikipedia.org.

There are two separate legends to assist in the Element Inspector.

Click on the Show Legend button to show the legend drawer, or click on the Hide Legend button to close the legend drawer
